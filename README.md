# Sanjoy Chattopadhyay Portfolio

Static portfolio site built with plain HTML, CSS, and JavaScript so it can be deployed directly on GitHub Pages.

## Files

- `index.html` - page structure and content
- `styles.css` - layout, themes, luxury dark styling, and animations
- `script.js` - theme toggle, reveal animations, metric counters, and card tilt

## Deploy On GitHub Pages

1. Create a GitHub repository, or use your existing `portfolio` repository.
2. Upload these files to the repository root.
3. Push the code to the `main` branch.
4. Open the repository on GitHub.
5. Go to `Settings -> Pages`.
6. Under `Build and deployment`, choose `Deploy from a branch`.
7. Select `main` and `/root`.
8. Save, then wait for GitHub Pages to publish the site.

If the repository name is `portfolio`, the site URL will usually be:

`https://sanjoy-chattopadhay.github.io/portfolio/`

## Content Notes

This version was written from public GitHub and LinkedIn profile details, then reframed to feel student-friendly and grounded instead of overly promotional.

## Customize Later

- Update any wording in `index.html`
- Adjust colors in the theme blocks inside `styles.css`
- Add or remove projects in the `#projects` section
- Replace the profile image URL if you want a different photo
